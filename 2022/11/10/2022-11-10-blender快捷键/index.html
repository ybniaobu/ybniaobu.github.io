Blender快捷键操作以及一些基础知识

发表于2022-11-10|更新于2023-07-02|blenderblender教程blender快捷键合集
|字数总计:8.7k|阅读时长:26分钟|阅读量:

注意事项

  • 快捷键要英文输入法！！！！！！
  • 有些快捷键要选中物体。
  • 区分小键盘的数字和数字键，小键盘会特别说明。
  • 最好的教程就是blender自己的英文版官方手册
  • Blender保存后打开会自动清理没有被使用的数据，可以使用伪用户来保存未使用的数据
  • 打包：将外部数据保存在blender工程中；解包：将blender工程数据保存在外部

基础操作

网格对象相关

物体模式下，移动、缩放、旋转等物体变化

  • Grab抓取状态（移动）：G
    • 再按X，锁定按x轴移动
    • 再按Y，锁定按y轴移动
    • 再按Z，锁定按z轴移动
    • 按2次X or Y or Z，会按物体自己的局部坐标移动，前提是当前坐标系为全局；若为当前为局部坐标系，按一次为局部，二次为全局
    • 按2次G，可以进行单一边线或顶点滑移
  • 同理缩放Scale、旋转Rotate：S、R，再对应X、Y、Z
    • 再按shift + X，锁定X轴，按Y、Z轴缩放（同理shift + Y、Z）
    • 可以输入数字，来设置缩放大小和旋转角度
    • 选择2个点按X或Y或Z轴方向对齐到2个点的中心，可以按S + X、Y、Z + 0
  • 重置移动、缩放、旋转：alt + G or S or R
  • 应用物体变化：ctrl + A；变换 -> 增量的意思就是改变“归零点”，应用过增量后，移动、缩放、旋转变换物体后，重置移动、缩放、旋转回归的是增量应用后的位置。此时若要重置回到世界原点，可以勾选清除增量
  • 镜像变换：ctrl + M，底部会有操作提示，可以按住鼠标中键拖拽

物体模式下，其他物体操作

  • 添加对象：shift + A
  • 复制对象：shift + D；关联复制alt + D即2个物体对应同一个mesh，改动一个，另一个也会变
  • 删除对象：X 或 Del
  • 物体合并join：ctrl + J，分离要进入编辑模式
  • 移动游标：shift + 鼠标右键
  • 调回添加物体设置菜单（微调面板）：F9
  • 侧边栏sidebar（n面板）：N
  • 工具栏：T可以隐藏/开启，小数字键盘的+ 或 -可以将UI放大缩小，也可以用鼠标拖拽让其并排显示
  • 坐标系选择菜单：,
  • 轴心点选择菜单：.
  • 开关吸附功能：shift + tab，或者在移动物体时，按住ctrl来临时开启吸附功能
  • 吸附上下文菜单：shift + S
  • 启用衰减编辑：O
  • 仅影响原点：ctrl + .
  • 重复上一步操作：shift + R
  • 建立父子级：选择多个物体按ctrl + P，活动项为父级，或者大纲视图（视图层）里把子级按住shift往父级拖
  • 清空父级：选择子级按alt + P，或者大纲视图（视图层）里把子级按住alt往外拖
  • 关联/传递数据：ctrl + L，根据最后的活动项关联
  • 多个物体同时操作：选中多个物体，按住alt，可以同时操作设置多个物体

编辑模式下，选择点边面

  • 编辑模式：Tab
    • 编辑模式下，按1为点模式
    • 编辑模式下，按2为线模式
    • 编辑模式下，按3为面模式
    • 按住shift再按1或2或3，可以同时选择多个模式（加选或减选）
  • 循环选择：alt + 鼠标左键
  • 选择并排边：ctrl + alt + 鼠标左键
  • 选择边界边：alt + 鼠标左键双击
  • 最短路径选择：ctrl + 鼠标左键
  • 反向选择（选择物体除了当前选择的部分）：ctrl + I
  • 加选或减选：ctrl + 小数字键盘的+ 或 -
  • 关联选择（选择相关联的所有点边面）：选中点、边、面 + L，鼠标要在选中的点、边、面附近，会受到划界的影响（详见微调菜单）；或者使用ctrl + L

编辑模式下，常用工具

  • 挤出选区Extrude Region：E，快速挤出：ctrl + 鼠标右键
    • 注意：按E后按右键取消，仍然会挤出，只不过重叠了看不见，需要撤销
    • 挤出上下文菜单：alt + E
  • 内插面Inset Faces：I
    • 上面会有操作提示
    • 再按住ctrl控制深度；再按O外插；多个面，再按I分别内插；若面存在边界（即边旁边没有面），再按B，可让边界边不内插。
  • 切割工具Knife：K，并按回车确认
    • 下面会有操作提示
    • 按住shift开启中点吸附，按住ctrl取消吸附
    • 按C可以开启切穿立体，最好在正交图下切穿，并且在下刀前开启
    • 在按钮模式下，关闭遮蔽几何体（上方），也可以切穿立体
    • 切割下的切分工具，可以讲立体一分为二，轴阈值可以让切线吸附至最近点或线
  • 倒角（斜切）Bevel：ctrl + B
    • 下面会有操作提示
    • 注意宽度类型：偏移量、宽度、深度、百分比和绝对的区别，斜切面的长度为宽度，原边到斜切面的边（垂直距离）为偏移量，原边到斜切面的距离为深度，百分比为临边的百分比，绝对为临边的绝对距离
    • 上下滚动鼠标滚轮，会增减斜切细分Segments的数量，即更平滑
    • 多细分的情况下，按P可以控制内凹还是外凸，即形状轮廓参数（默认0.5为圆）
    • 微调菜单中硬化法向可以平滑倒角的同时不影响原始面法向
    • 外斜接的补块和圆弧以及内部的圆弧可以更优化处理多条连接的线的倒角导致的连接问题；相交类型自动的栅格填充会自动填充多个相交边同时倒角产生的面，截止则会不产生该面
    • 轮廓类型的自定义里面包括阶梯类型等等
    • 一般为边倒角，顶点倒角为ctrl + shift + B，可以使用该方法来给平面中心开洞（配合球形化工具），或者让曲线变的圆滑
    • 建议的倒角段数为2的倍数，比如4
  • 环切Loop Cut（环切并滑移）：ctrl + R
    • 上下滚动鼠标滚轮，会增减环切边的数量
    • 上面会有操作提示
    • 输入数字，来设置环切边的数量
    • 微调菜单中的均匀和翻转指的是环切的相邻的边非平行情况下使环切线优先匹配其中一条边
    • 偏移环切边：ctrl + shift + R，和倒角类似，但是若选中的边非直线，偏移环切边和环切模式类似，新生成的线是选中的边和它平行的边的平均化后的线（使用均匀或翻转可以改变该模式），而倒角则完全是原来选中的边的样子。微调菜单中还有个封盖端点的功能
  • 多边形建形工具Poly Build：选择此工具后
    • 当编辑模式下，没有点时，按住ctrl + 左键单击，可以生成点，可以按住左键拖动点
    • 当存在面后，该工具会预选择或者说是捕捉到一条线（蓝色的线）按住ctrl + 左键单击，会形成三角形，再次按住ctrl，形成三角形，2个三角形会形成四边形。也可以按住左键拖动，挤出边（四边形）
    • 按住shift + 左键可以快捷删除线（四边形面）或点（三角形）
  • 旋绕工具Spin：以3D游标为中心，按圆形挤出
    • 旋转方向的不同会改变法线的方向，需注意法线的方向
  • 光滑工具Smooth
    • （光滑）平滑顶点：当有多个顶点时，微调菜单中的重复次数可以增加所影响的面（因为一次只影响相邻顶点）
  • 滑移工具Slide
    • 双击G可以对单一边线或者单一顶点进行滑移，操作提示在上方
    • 顶点滑移：可以选取多个顶点并按shift + V进行滑移
  • 法向收缩：alt + S
    • 法向收缩基于选择对象的法向进行缩放，而普通的S缩放基于轴心点缩放
    • 推/拉：等距离的缩放，S缩放是等比例
  • 切变工具Shear
    • 切变以及球形化，球形化可以结合顶点倒角进行面开洞
  • 断离工具Rip：V
    • 根据鼠标的位置系统自动确定断离的方向
    • 注意：按V后按右键取消，仍然会断离，只不过重叠了看不见，需要撤销
    • 断离并进入滑移：V + G
    • 断离边线（挤出顶点并移动）：alt + D，可以快速加点，再按G可以进入滑移状态

编辑模式下，其他操作

  • 选择菜单下的操作：
    • 全选：A，弃选：alt + A
    • 框选：B，刷选：C，按住shift拖动鼠标取消选取
    • 反转（反向选择）：ctrl + I
    • 选择相似菜单：shift + G
    • 有些操作见：编辑模式下，选择点边面
  • 网格菜单下的操作：
    • 重置蒙皮尺寸（即缩放蒙皮半径）：ctrl + A，见蒙皮修改器
    • 吸附上下文菜单：shift + S
    • 合并上下文菜单：M，自动合并顶点在右上角开启
    • 拆分上下文菜单：alt + M；拆分选中项：Y
    • 物体分离Separate：P，变为独立的物体
    • 凸壳：根据物体生成罩体
    • 法向上下文菜单：alt + N；法向旋转可以用R + N
    • 清理菜单：精简几何体可以减少面数；简并融并可以融并面积为零的面或长度为零的边，可以去除重叠的点；有限融并根据夹角来融并点和线；平整表面可以把不可能的四边形恢复水平
  • 顶点菜单下的操作：
    • ctrl + V弹出顶点菜单，V for vertex
    • 从顶点创建边/面：F；连接顶点路径：J；注意F和J的区别，F是架桥，不影响沿途边线（相当于新连接的线会与沿途的边线重叠在一个面上）；J是修路，会切断沿途边和面，在一个面上使用连接顶点建议使用J，但是J无法连接彼此完全分离的点
    • 打开顶点组菜单：ctrl + G
    • 钩挂：ctrl + H，详见钩挂修改器
  • 边菜单下的操作：
    • ctrl + E弹出边菜单，E for edge
    • 桥接循环边，或者搜索Bridge Edge Loops（需2组循环边）
    • 边线折痕强度：shift + E并拖动
  • 面菜单下的操作：
    • ctrl + F弹出面菜单，F for face
    • 尖分面：将面拆分为数个三角面，会新建一个中心点；面三角化：ctrl + T将面拆分为数个三角面，不会新建点
    • 三角填充：alt + F
    • 交集（切割）与交集（布尔）为自带的布尔运算
  • 快速收藏夹：Q，可以在想添加按钮右键选择添加到快速收藏夹，之后就可以使用这个快捷使用

曲线对象相关

曲线编辑模式下相关操作

  • 曲线的样条类型：①多段线：控制点连线为固定的直线；②贝塞尔；③NURBS(非均匀有理B样条)
  • 可以在视图叠加层开启全部控制柄，这样可以不先选中控制柄再调节，加快效率
  • 贝塞尔曲线：控制点拥有权重、半径和倾斜属性（见N面板）
    • 自由线画笔工具：表(曲)面可以让线吸附至网格表面；锥化可以控制曲线控制点半径；偏移量指从表面偏移画笔，未开启绝对偏移时，偏移值为系数*半径
    • 控制柄切换：V；对齐（粉色）：同控制点控制柄协同操控；自由（红色）：单独操控；自动（黄色）：转化为均值状态并自动适应，手动介入手柄会使其转换为对齐类型；矢量（绿色）：基于点连线，手动介入手柄转换为自由类型。重新计算控制柄：shift + N
    • 挤出工具：E；快速挤出：ctrl + 鼠标右键
    • 调节半径：alt + S
    • 调节倾斜：ctrl + T，倾斜控制曲线法向的方向；重置倾斜：alt + T
    • 切换循环alt + C将曲线变为闭合状态；或者直接F首尾相连；或者在曲线属性的活动样条线里勾选循环
  • NURBS(非均匀有理B样条)的每个控制点有个W值（见N面板），即weight，其根据控制点和W值进行曲线生成

视角、视图、窗口相关

改变视角

  • 旋转视角：鼠标中键
  • 缩放视角：鼠标滚轮 或者 ctrl + 鼠标中键
  • 平移视角：shift + 鼠标中键
  • 快速聚焦：小数字键盘上的·，或者按住~选择View Selected
  • 快速回归全场景（聚焦全场景）：Home
  • 隐藏物体：H
  • 隐藏所有除选中物体：shift + H
  • 取消所有隐藏：alt + H
  • 第一人称视角：shift + ~，底部会有操作提示

视图快捷键

  • 正交前视图：小数字键盘1；后视图：ctrl + 小数字键盘1
  • 正交右视图：小数字键盘3；左视图：ctrl + 小数字键盘3
  • 正交顶视图：小数字键盘7；底视图：ctrl + 小数字键盘7
  • 以上直接使用右上角xyz的那个坐标轴比较方便
  • 视角对齐物体局部坐标的xyz轴：shift + 小数字键盘1或3或7
  • 视图反转：小数字键盘9
  • 透视图和正交视图的切换：小数字键盘5
  • 四格视图：ctrl + alt + Q
  • 局部视图（只显示选中的物体，可以多选）：/，再次点击退出局部视图
  • 最大化视窗（窗口）：ctrl + 空格，根据鼠标的位置选择最大化的窗口，再次点击退出；全屏视窗：ctrl + alt + 空格，同上
  • 视图着色方式菜单：Z；透视模式：alt + Z，透视模式 + 线框模式：shift + Z

摄像机相关

  • 切换到摄像机视角：小数字键盘的0，摄像机视图大小，受 Output properties 输出属性的 Format 格式中 Resolution 分辨率控制，默认1920px*1080px
  • 摄像机镜头对齐到当前屏幕视角：ctrl + alt + 小数字键盘的0，或者N面板上的view，选择Camera to View
  • 切换到摄像机视角后，并且Camera to View（锁定摄像机到视图方位），可以开启第一人称视角（shift + ~）来移动（微调）摄像机

其他名词解释

  • 缝合边Seam：UV展开的边；锐边Sharp：自动平滑不影响的边；Freestyle边：模型的轮廓边（比如三渲二下模型的描边效果）
  • 形态键Shape Keys：用于记录顶点排布方式
  • 纹理空间Texture Space：纹理图案映射到三维物体表面的区域，可以通过改变纹理空间配合UV改变纹理的显示区域和效果
  • 平滑着色Shade Smooth：改变了面的法向的方向从而改变了表面着色，造成光滑表面的错觉（可以平滑着色后看看法线方向的变化）。法线与着色相互关联，完全拆分拆分法向（可以在视图叠加层显示拆分法向）可以模拟锐利的棱（即锐边），完全合并拆分法向可以模拟平滑。
  • 纹理基础：
    • 图像：RGB三条通道混合产生颜色，通道位宽（bit）影响精度，而每条通道位宽可以以灰度信息进行存储，比如8 bit（0-255）为例，0即黑色对应通道贡献为0，255即白色对应通道贡献为1；而Alpha通道存储透明信息，黑色即alpha为0，显示为绝对透明，白色即alpha为1，显示为绝对不透明
    • 高动态范围图像 HDRI 或 HDR (High Dynamic Range Imaging)，比普通8 bit RGB图像拥有更大的亮度范围，HDR文件通常使用32位浮点数来存储每个像素的颜色值；EXR文件使用16位或者32位浮点数格式
    • Alpha直通型：RGB先混合再乘Alpha；预乘型：RGB分别乘Alpha再混合；主要因为sRGB是非线性色彩空间，所以直通型和预乘型会有不同效果；通道打包：Alpha不与RGB通道相乘，互不影响，因为RGBA包含4个灰度通道，可以在每个通道里存储不同的灰度信息，从而1张图可以当4个灰度图使用，从而节约内存空间

修改器相关操作

修改器的优势在于其操作可逆，更能满足动画的需求

通用修改器操作

  • 批量应用修改器：物体模式下，ctrl + A，选择可视几何->网格；或者使用官方插件（界面：Modifier Tools 修改器工具），可以直接在修改器上面全部应用
  • 在一个物体内复制修改器：鼠标放在要复制的修改器上shift + D
  • 复制/传递修改器至其他物体：物体模式下，ctrl + L，选择复制修改器（注意复制的是最后选择的活动项的修改器）；或者使用官方插件（界面：Copy Attributes Menu 复制属性菜单），选择要被复制的物体，加上最后选择的提供修改器属性的物体，按ctrl + C选择复制修改器；或者使用修改器的下拉菜单（同上最后选择活动项）点击复制到选定项
  • 大纲视图里也可以改变修改器的顺序；也可以在大纲里把修改器拖至其他物体，实现复制/传递修改器的效果
  • 注意：法向会影响到修改器的效果

生成类 generate 修改器

生成修改器能够影响整个网格的拓扑，侧重处理原始网格生成新的结构，往往伴随着点边面的数量变化

阵列修改器 Array

  • 适配类型：根据数量、长度或者曲线（曲线的长度，与形状无关）来复制网格物体
  • 相对偏移 = 原始网格边界框*系数；恒定偏移 = 质心点距离；物体偏移根据物体与参考物体质心点的距离偏移，同时可以传递缩放和旋转信息（比如在物体模式下对参考物体缩放），其实本质是传递位置、旋转和缩放的信息
  • 合并选项可以合并距离小于设定值的顶点，注意可能会保留的内部面
  • UV选项可以让UV进行偏离，根据贴图的U或V方向偏离地显示一定区域
  • 可以借助父级网格顶点、面来生成实例，实现类似阵列修改器的效果；也可以用实例化配合阵列修改器，让除网格外的物体也能形成阵列效果（阵列修改器只作用于网格物体）；实例化类似于面向对象的实例化对象的概念，实例化集合类似于引用，可以减少内存压力
  • 也可以借助网格生成粒子毛发，将毛发替换为物体来实现类似阵列修改器的效果

倒角修改器 Bevel

  • 与编辑模式下的倒角工具类似，详见上面
  • 限定方式选项的角度、权重以及顶点组可以对倒角的范围进行限制

布尔修改器 Boolean

  • 两种解算器：①快速：传统的布尔算法，速度快，但是对重叠孔洞等非流形表现不佳；②准确：更高准确度，但面数较多时需要更多算力，可能会卡顿
  • 可以使用官方插件（物体：Bool工具，建议勾选快速转换和display as wireframe）来代替该修改器提高工作效率：被布尔的物体为画布，指定目标物体为笔刷，先选择笔刷，再选择被布尔的画布。Brush布尔是添加布尔修改器但是不应用，自动布尔直接应用修改器。快速转换勾选上，点击笔刷物体，开启快速可视化，可以在准确的解算器模式下移动笔刷物体而不卡顿（其实就是移动时暂时关闭了修改器的视图实时显示）。ctrl + shift + B可以跳出布尔工具菜单
  • 法向会影响布尔的效果，笔刷物体的法向（着色）信息会传递到被布尔的物体，注意面朝向导致的布尔异常

建形修改器 Build

  • 使网格物体根据时间推移一个面一个面得建立或消失，顺序跟网格元素排序有关
  • 结合物体实例化、粒子系统来实现比如灯光等等的出现与消失

精简修改器 Decimate

  • 用于减少面与顶点的数量，并尽量维持物体的结构
  • 有三种算法：塌陷、反细分和平面。塌陷：渐进式地合并顶点；反细分：可以认为是反向的细分，它试图删除细分操作的结果；平面：减少角度低于角度限制的平面来精简面数

拆边修改器 Edge Split

  • 将满足条件参数的边一分为二地拆开

遮罩修改器 Mask

  • 借助顶点组或骨架，对网格物体的网格进行遮蔽隐藏

镜像修改器 Mirror

  • 镜像中心默认为物体自身原点，可设置为其他物体
  • 使用该修改器时，注意内部面的产生

表面细分修改器 Subdivision Surface

  • 添加表面细分修改器：物体模式下，ctrl + 1，数字1-5都可以，代表的是视图层级的数字
  • 简单型就是对每个面进行简单细分，不影响面与面之间的角度；Catmull-Clark算法会平滑网格
  • 优化显示需要在物体模式下开启线框，开启优化显示，会显示原来边线的变化，不包括内部的细分边线

多级精度修改器 Multiresolution

  • 跟表面细分修改器类似，但是在雕刻的时候使用该修改器合适，在雕刻模式下，只有多级精度修改器的处理数据支持直接被雕刻。使用表面细分修改器后开启雕刻模式，雕刻的是原始的基本网格，而不是被修改器修改的网格
  • 多级精度修改器之前不能有其他修改器，它需要在修改器堆栈顶部
  • 多级精度修改器可以一层一层添加，每一层可以用不同的算法。细分：默认的Catmull-Clark算法；简单型：对原始网格的边进行插值；线性：对目前网格物体的面进行简单细分，不影响目前网格物体的曲率
  • 物体细分应用后，可以添加多级精度修改器进行反细分

重构网格修改器 Remesh

  • 基于当前网格重新计算生成拓扑结构，拓扑结构只包含四边形
  • 包括4种模式，块状Blocks：把原网格变成积木；平滑化Smooth：重新拓扑为只包含四边形的拓扑结构；锐利Sharp：类似于平滑化，但保留锐边和转角；体素Voxel：使用OpenVDB从当前的几何体生成一个新的流形网格，同时尝试保留网格的原始体积，体素大小越小模型越精细
  • 可以使用物体数据属性中的重构网格来代替该修改器，需要点击下面的体素重构或者网格重构；在雕刻模式下，体素网格重构（相当于按了重构网格按钮）：ctrl + R；体素大小预估：R；四边形网格重构：ctrl + alt + R

螺旋修改器 Screw

  • 螺旋轴心是物体的原点
  • 注意可能产生的重叠点，别忘了勾选合并

蒙皮修改器 Skin

  • 根据点和边线（与面无关）生成蒙皮网格数据，可用于制作基本形状，为雕刻提供便利
  • 在编辑模式下的点模式，选择一个顶点，使用ctrl + A可以控制蒙皮半径，即n面板中的顶点数据的半径X和半径Y，可以再点击X或Y控制半径X和半径Y
  • 分支平滑：可以控制当一个点有多个分支时该点附近蒙皮效果的半径（不调整会过于厚实）
  • 根点：带着红圈的顶点，为蒙皮修改器不可或缺的元素，删除会导致故障

实体化修改器 Solidify

  • 实体化厚度增加减少的方向与法向有关，偏移量为1时，且厚度乘以偏移量为正数，厚度向法向的正方向增加；偏移量为-1，且厚度乘以偏移量为负数，厚度向法向的负方向增加；偏移量为0，厚度均匀地往法向正负方向增加
  • 复杂模式可以处理非流形图形（一条边连接2个以上的面），而简单型模式会出现错误
  • 对于厚度异常，可以先选择均衡厚度解决；若不行，再选择法向里的高质量来解决。遇到该问题，处理的顺序可按下面，越后面处理能力越强，相对计算时间也越长：①简单模式或复杂模式+固定；②简单模式+均衡厚度或者法向里的选项；③简单模式+均衡厚度和法向里的选项或者复杂模式+均匀；④复杂模式+约束

三角化修改器 Triangulate

  • 将四边面或多边面转换为三角面

焊接修改器 Weld

  • 修改器版本的按间距合并
  • 相连项模式仅仅合并同一条边上（具有相连关系的顶点），全部模式合并所以满足条件的顶点
  • 注意顶点合并了，但是可能会产生内部面

线框修改器 Wireframe

  • 将平面变为线框并加厚
  • 偏移量和厚度与实体化修改器类似

形变类 deform 修改器

形变类修改器只改变对象的形状，不改变其拓扑结构，大多数变形基于物体原点变形。比生成类修改器多出应用为形态键以及保存为形态键，保存为形态键不会应用修改器，即点击完修改器仍然存在，而应用为形态键点击完后修改器应用掉了。注意：因为只改变对象形状，所以添加修改器的物体需要足够多的顶点，否则很难呈现效果

铸型修改器 Cast

  • 变形为球形、柱体或者方体，球形类似于球形化工具
  • 变形默认以物体原点为变形中心，可以尝试改变原点看看
  • 轴向可以控制在各自轴向上是否变形；半径可以控制以物体原点为中心的变形范围，默认的0为不产生限制效果；尺寸调节发生形变部分的网格的大小尺寸；物体可以设置其他物体的原点为铸型效果中心，默认仅位置，可以勾选使用变换

曲线修改器 Curve

  • 使用曲线物体折弯网格，同理变形默认以物体原点为变形中心
  • 修改器修改的是网格数据，并非原点数据。当网格物体与曲线原点重合（没有相对位置差和旋转差），将会使用网格原点位置（变形原点）对应曲线起始点，所以若变形原点两侧都有网格，网格变形后的一部分会在曲线外面
  • 前进轴取决于形变轴设置与物体原点间的旋转差，推荐使用时避免旋转差；网格原点位置和曲线原点有位置差，那么变换后的网格原点位置与曲线起始点的位置也会有该位置差（基于若该修改器会修改网格原点的局部坐标）
  • 曲线的性质也能影响到变形，比如半径、倾斜等等

置换修改器 Displace

  • 基于纹理偏离顶点，对8比特位图非矢量置换来说，纯黑0对应纹理强度0，纯白255对应纹理强度1
  • 强度力度控制纹理强度；中间值与实体化修改器的偏移量类似，控制位移

钩挂修改器 Hook

  • 利用其他物体控制网格或曲线的顶点（需要指定，指定需进入编辑模式）从而改变形状
  • 区分指定的参与钩挂的顶点和顶点组，同时顶点组权重驱动强度的范围为参与钩挂的顶点（即指定的顶点）
  • 衰减中心为物体原点

拉普拉斯形变修改器 Laplacian Deform

  • 一种积分变换，对非锚点顶点执行自适应计算以尽可能保持原始几何形态；使用必须指定锚点权重，然后绑定，然后在编辑模式移动指定锚点的一些顶点，修改器将其余锚点顶点保持在固定位置，并计算所有剩余顶点的最佳位置以保留原始几何细节
  • 建议指定锚点时，一些为固定点，一些为移动点，或者这两部分都钩挂到另外物体上

晶格形变修改器 Lattice

  • 通过给网格添加外部晶格控制物体（需绑定），借助插值计算实现更少点控制更多点的操作
  • 建议在物体模式下将晶格物体缩放适配被变形物体，建议设置晶格物体为父级，建议使用形态键记录晶格顶点初始位置
  • 晶格物体的细分在物体属性里的分辨率设置以及插值类型（可以改变形变的算法）
  • 可以通过设置父级目标ctrl + P中的晶格形变也可以对物体添加该修改器

网格形变修改器 Mesh Deform

  • 与晶体形变修改器相似，用外部网格物体控制物体形变（需绑定）
  • 网格笼注意事项：尽可能包裹物体（在网格外就不会形变）；面数不宜过多；设置线框显示以及渲染不可见；法向朝外；绑定父级
  • 相对于晶格，网格笼的初始态取决于绑定时的样子
  • 对于面数很多的物体要生成网格笼，可以复制物体并生成凸壳（网格菜单里）再通过网格重构处理，从而减少面数
  • 可以使用形态键控制网格笼从而以小控制大的变形，可以减少计算量

缩裹修改器 Shrinkwrap

  • 将投射物体的顶点缩裹至目标物体
  • 注意事项：缩裹形变是作用于物体顶点的，需要足够的顶点支撑变形后的结构；建议尽可能让缩裹物体与目标形态一致
  • 缩裹方法：①最近表面的位置点：顶点投射到离目标物体表面的最近的点，最近的点不一定是垂直位置的点；②投影（允许2个目标物体）：根据被修改的物体的法向或轴向推移顶点直到与目标接触，若无法接触的顶点则位置不变；③最近的顶点：顶点投射到离目标物体表面的最近的顶点（网格上已存在的顶点）；④目标法线投影：使用目标物体的插值法向（自动平滑法向）
  • 吸附方式：①表面上：不受目标物体法向影响的偏移；②向内/向外：目标法向向外，正值实现向内/向外偏移；③表面外围：正值向法向外侧偏移；④表面上方：沿目标物体法向使用偏移距离
  • 可以借助缩裹实现表面重新拓扑

简易形变修改器 Simple Deform

  • 扭曲：顶点按所选旋转轴螺旋排布（基于原点的旋转轴）
  • 弯曲：根据所选轴向旋转形变，选择x、y沿着z轴弯曲，选择Z沿着x轴弯曲（看似不直观，但开关修改器结果就直观了）
  • 锥化：原点决定锥化中心，沿着所选轴向锥化
  • 拉伸：原点决定拉伸中心，沿着所选轴向拉伸；拉伸可以借助空物体的缩放进行调节
  • 注意事项：注意首尾合并，不会自动合并顶点
  • 可以使用晶格来间接控制物体，然后给晶格简易形变修改器

平滑修改器 Smooth

  • 不改变顶点数平滑网格，系数建议为0-1

矫正平滑修改器 Smooth Corrective

  • 矫正平滑依据某个初始状态，将当前顶点状态朝向初始状态平滑，有利于减少网格高度扭曲
  • 在该修改器之前添加生成类修改器可能会导致失效需要重新绑定坐标（静置源中处理）

拉普拉斯平滑修改器 Smooth Laplacian

  • 减少网格表面的起伏噪点，特别是对扫描模型很有用
  • Lambda系数控制面平滑，Lambda边界控制边界平滑，规格化用于避免极端数据的出现

表面形变修改器 Surface Deform

  • 相对于晶格形变和网格形变的包裹式形变，表面形变修改器可以根据任意网格曲面控制，本质是传递其运动/形变信息
  • 同理，可以以小控制大

弯绕修改器 Warp

  • 需要两个物体，从源物体变形至目标物体
  • 可以基于纹理灰度图变形，比置换修改器方便控制范围和衰减

波浪形变修改器 Wave

  • 理解为沿着Z轴或法向上下移位比较好
  • 生成单向线性震荡波或单向圆形波（XY控制），会自动添加动画（循环不开启就一次）
  • 顶点沿自身Z轴（可以改为沿法向）实现周期型移位，沿法向可以产生变大变小的动画效果（类似于心脏搏动）
  • 宽度为半周期长度，窄度越高，脉冲越窄
  • 时间：0帧为起始帧，速度正则波形向外扩散，速度负波形向内收缩（把偏移量拉大就可以看到）；阻尼是额外数量的帧，波从高度衰减至0；建议速率为周期（（宽度*2）/动画帧数）的倍数，否则动画会出现断离，因为没法满足循环
  • 可以基于纹理上下移位

动画相关操作

关键帧

  • 插入关键帧：I
  • 平滑关键帧：alt + O
