# 第一章 基础
## C++ 程序
C++ 融合了 3 种编程方式：① C 语言代表的**过程性语言**；② C++ 代表的**面向对象语言**，通过在 C 语言上添加类；③ C++ 模板 template 支持的**泛型**编程。因此 C++ 是 C 语言的超集。


C++ 程序 program 运行起来的基本步骤或要素有如下：  
①**源代码 source code**：使用文本编辑器编写的文件，这些文件组成了程序的源代码；  
②**编译与目标代码 compile & object code**：编译 compile 意味着运行一个程序将源代码翻译为**机器码 machine language**，从而能够被其他主控电脑使用。而包含了这个翻译后程序的文件就是你的程序的目标代码 object code；  
③**链接 Link**：将目标代码和其他代码链接起来，C++ 通常会使用**库 library**，库包含了一系列函数的目标代码。而链接就是将你的程序的目标代码和使用到的函数的目标代码联合一些标准启动代码生成一个可以运行的版本。包含该最终产品的文件称为**可执行代码 executable code**。
